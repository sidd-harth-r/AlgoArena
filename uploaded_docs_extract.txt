0. What You Are Building
AlgoArena is a web-based competitive programming platform — think LeetCode, but with one feature that does not exist anywhere else: after you submit code, the platform infers the Big-O time complexity of your solution using static AST analysis (not runtime benchmarking) and passes that complexity gap to an AI that generates a Socratic hint. The hint does not tell you the answer. It asks you the question that leads you there.
0.1 The Three Layers of Feedback
Layer 1 — Execution feedback (every judge has this): Pass / fail per test case, runtime in ms, error type (TLE, MLE, compile error, runtime error).
Layer 2 — Complexity feedback (our differentiator): Static Big-O inference from the submitted AST. No other public judge does this.
Layer 3 — Socratic AI hint (our differentiator): Complexity gap fed to Claude. The model is constrained to ask a question, not give the answer.
0.2 What You Are NOT Building
Scope creep is how interns fail. The following are explicitly out of scope for this internship. Do not build them.
A leaderboard or ranking system
A DP-based personalised learning path scheduler
AI-generated test cases (unreliable without human review)
Multi-language AST analysis (Python only for v1)
A user study or analytics pipeline
Social features, comments, or discussion threads

12. Common Mistakes — Read Before You Start
Mistake 1: Blocking the FastAPI event loop
If you call Judge0 synchronously inside an async function using requests.get() instead of httpx.AsyncClient, every submission will block the entire server. Use async/await everywhere inside async def functions.
Mistake 2: Storing secrets in git
Your .env and .env.local files must be in .gitignore before the first commit. Run: echo '.env' >> .gitignore && echo '.env.local' >> .gitignore before git add.
Mistake 3: Polling without a timeout
Your frontend polling loop must have a maximum retry count (e.g., 20 retries x 1.5s = 30s timeout). Without it, a broken submission will poll forever and drain the user's battery.
Mistake 4: Tree-sitter grammar version mismatch
tree-sitter and tree-sitter-python must be compatible versions. Pin them: tree-sitter==0.21.3 and tree-sitter-python==0.21.0 in requirements.txt. Mixing major versions breaks the Language() constructor.
Mistake 5: SSE connection not closed on component unmount
In AiTutorPanel, the AbortController ref must call abort() in a useEffect cleanup function. Without it, navigating away from a problem page while a hint is streaming will cause React state updates on unmounted components.
Mistake 6: Test case format inconsistency
Your seed script must produce test case inputs in exactly the format the submitted code expects. For Two Sum, the input is two lines: the array on line 1, the target on line 2. Document this format in a comment next to each problem's test cases.
Mistake 7: Modifying the AI prompt without approval
The Socratic constraint in the system prompt is the product's intellectual contribution. If you modify it to 'be more helpful' by giving more direct answers, you break the product. Do not change the prompt without discussing it first.

Priority | Pattern Detected | Inferred Complexity
1 (highest) | A loop nested 3+ levels deep (for inside for inside for) | O(n^3)
2 | A recursive function with 2 recursive calls (no @lru_cache) | O(2^n)
3 | A loop nested exactly 2 levels deep (for inside for) | O(n^2)
4 | A call to sorted() or .sort() inside no loop | O(n log n)
5 | A single for or while loop (no nesting, no recursion) | O(n)
6 | A recursive function with @lru_cache decorator | O(n)
7 (fallback) | No loops, no recursion detected | O(1)

      p("This is the most important file in the backend. Read it carefully. Every submission goes through this exact sequence."),
      codeBlock([
        "# apps/api/routers/submissions.py",
        "from fastapi import APIRouter, Depends, HTTPException, BackgroundTasks",
        "from sqlalchemy.orm import Session",
        "from models.database import get_db",
        "from models.models import Submission, Problem, TestCase, SubmissionStatus, BigOClass",
        "from services.judge import submit_to_judge0, poll_judge0",
        "from services.complexity import infer_complexity",
        "from pydantic import BaseModel",
        "import uuid",
        "",
        "router = APIRouter()",
        "",
        "class SubmitRequest(BaseModel):",
        "    problem_id: int",
        "    code: str",
        "    language: str   # 'python', 'javascript', 'cpp'",
        "",
        "@router.post('/')",
        "async def create_submission(",
        "    req: SubmitRequest,",
        "    background_tasks: BackgroundTasks,",
        "    db: Session = Depends(get_db),",
        "):",
        "    problem = db.query(Problem).filter(Problem.id == req.problem_id).first()",
        "    if not problem:",
        "        raise HTTPException(status_code=404, detail='Problem not found')",
        "",
        "    # Create PENDING submission record immediately",
        "    sub = Submission(",
        "        id=uuid.uuid4(),",
        "        problem_id=req.problem_id,",
        "        user_id='00000000-0000-0000-0000-000000000001',  # hardcode until auth is wired",
        "        code=req.code,",
        "        language=req.language,",
        "        status=SubmissionStatus.PENDING,",
        "        total_count=len(problem.test_cases),",
        "        optimal_complexity=problem.optimal_complexity,",
        "    )",
        "    db.add(sub)",
        "    db.commit()",
        "    db.refresh(sub)",
        "",
        "    # Run evaluation in background so we can return the submission ID immediately",
        "    background_tasks.add_task(evaluate_submission, str(sub.id), req.code, req.language, problem, db)",
        "    return {'submission_id': str(sub.id), 'status': 'PENDING'}",
        "",
        "async def evaluate_submission(sub_id, code, language, problem, db):",
        "    sub = db.query(Submission).filter(Submission.id == sub_id).first()",
        "    sub.status = SubmissionStatus.RUNNING",
        "    db.commit()",
        "",
        "    test_cases = db.query(TestCase).filter(TestCase.problem_id == problem.id).all()",
        "    passed = 0",
        "    failed_input = failed_expected = failed_actual = error_msg = None",
        "",
        "    for tc in test_cases:",
        "        result = await submit_to_judge0(code, language, tc.input_data, problem.time_limit_ms)",
        "        judge_status = result.get('status', {}).get('id')",
        "",
        "        if judge_status == 3:   # Accepted",
        "            if result.get('stdout', '').strip() == tc.expected_out.strip():",
        "                passed += 1",
        "            else:",
        "                if not failed_input:",
        "                    failed_input    = tc.input_data",
        "                    failed_expected = tc.expected_out",
        "                    failed_actual   = result.get('stdout', '')",
        "        elif judge_status == 5:   # TLE",
        "            sub.status = SubmissionStatus.TIME_LIMIT ; break",
        "        elif judge_status == 6:   # MLE",
        "            sub.status = SubmissionStatus.MEMORY_LIMIT ; break",
        "        elif judge_status in [11, 12]:  # Runtime error",
        "            sub.status    = SubmissionStatus.RUNTIME_ERROR",

        "# apps/api/services/complexity.py",
        "from tree_sitter import Language, Parser, Node",
        "import tree_sitter_python as tspython",
        "from models.models import BigOClass",
        "",
        "PY_LANGUAGE = Language(tspython.language())",
        "_parser = Parser(PY_LANGUAGE)",
        "",
        "def infer_complexity(code: str) -> BigOClass:",
        "    tree = _parser.parse(code.encode())",
        "    root = tree.root_node",
        "    max_loop_depth = _max_loop_depth(root, 0)",
        "    has_recursion, has_memo, branch_count = _check_recursion(root)",
        "    has_sort = _has_sort_call(root)",
        "",
        "    if max_loop_depth >= 3:                          return BigOClass.ON3",
        "    if has_recursion and not has_memo and branch_count >= 2: return BigOClass.O2N",
        "    if max_loop_depth == 2:                          return BigOClass.ON2",
        "    if has_sort and max_loop_depth == 0:             return BigOClass.ONlogN",
        "    if max_loop_depth == 1 or (has_recursion and has_memo): return BigOClass.ON",
        "    return BigOClass.O1",
        "",
        "def _max_loop_depth(node: Node, current_depth: int) -> int:",
        "    \"\"\"Recursively walk the AST counting max nested loop depth.\"\"\"",
        "    if node.type in ('for_statement', 'while_statement'):",
        "        current_depth += 1",
        "    max_depth = current_depth",
        "    for child in node.children:",
        "        max_depth = max(max_depth, _max_loop_depth(child, current_depth))",
        "    return max_depth",
        "",
        "def _check_recursion(root: Node):",
        "    \"\"\"",
        "    Returns (has_recursion, has_lru_cache, branch_count).",
        "    has_recursion: True if any function calls itself.",
        "    has_lru_cache: True if any function is decorated with @lru_cache or @cache.",
        "    branch_count: max number of recursive calls in one function body.",
        "    \"\"\"",
        "    results = []",
        "    _walk_functions(root, results)",
        "    if not results:",
        "        return False, False, 0",
        "    has_rec   = any(r[0] for r in results)",
        "    has_memo  = any(r[1] for r in results)",
        "    max_calls = max(r[2] for r in results)",
        "    return has_rec, has_memo, max_calls",
        "",
        "def _walk_functions(node: Node, results: list):",
        "    if node.type == 'function_definition':",
        "        fn_name = next((c.text.decode() for c in node.children if c.type == 'identifier'), None)",
        "        decorators = [c for c in node.children if c.type == 'decorator']",
        "        decorator_names = [d.text.decode() for d in decorators]",
        "        has_memo = any('lru_cache' in d or '@cache' in d for d in decorator_names)",
        "        recursive_calls = _count_recursive_calls(node, fn_name)",
        "        results.append((recursive_calls > 0, has_memo, recursive_calls))",
        "    for child in node.children:",
        "        _walk_functions(child, results)",
        "",
        "def _count_recursive_calls(fn_node: Node, fn_name: str) -> int:",
        "    count = 0",
        "    if fn_node.type == 'call':",
        "        func = fn_node.child_by_field_name('function')",
        "        if func and func.text.decode() == fn_name:",
        "            count += 1",
        "    for child in fn_node.children:",
        "        count += _count_recursive_calls(child, fn_name)",
        "    return count",
        "",
        "def _has_sort_call(root: Node) -> bool:",
        "    if root.type == 'call':",
        "        func = root.child_by_field_name('function')",
        "        if func:",
        "            text = func.text.decode()",
        "            if text == 'sorted' or text.endswith('.sort'):",
        "                return True",
        "    return any(_has_sort_call(c) for c in root.children)",
      ]),
